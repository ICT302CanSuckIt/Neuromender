Changing Game settings:

First you must navigate to the profile of the patient who's settings you wish to change navigate to the reports page as shown previously the select View all user profiles as the report type and click submit.

When you reach the View all user profiles report page click on the name of the patient who's setting you wish to change.

This will take you to the Patients profile page. click edit to start changing the patient's game settings.

Wingman Settings:

Scroll down the page to find Wingman Game Settings if it is not enabled it will look like the picture below, click the box next to Enable Wingman Game to bring up the game settings.

Once enabled the game settings will look like this:

The table below gives a discretion of each setting

Field

Description

Angle Threshold

The highest angle (in degrees) to which game will allow the survivor to lift their arm

Threshold Increment-Decrement

By how much the angle threshold (in degrees) will increase or decrease in response to survivor gameplay

Track Slow

The length (in seconds) of the slow track mode

Track Medium

The length (in seconds) of the medium track mode

Track Fast

The length (in seconds) of the fast track mode

Max games per session

The maximum number of games that can be played for each session (time between login/logout)

Max games per day

The maximum number of games that can be played during an entire day

Interval between session

The minimum number of hours after a session before a survivor can play again

Make sure Enter a value for each setting. you can use the up and down arrow shown in the picture above or you can click on the field you wish to change and type in the desired value. once all changes to the settings are finished click commit changes.

Targets settings

Scroll down the page to find Targets Game Settings if it is not enabled it will look like the picture below, click the box next to Enable Targets Game to bring up the game settings.

Once enabled the game settings will look like this:

Changing the grid size will automatically update the size of the grid shown on the page and clear the grid order.

clicking a panel in the grid will show a box under the number of the panel signifying how many times it has been clicked, A number will also show up in the top left of the panel signifying when in the grid order it appears this also updates the grid order. Clicking the number in the top left of the grid panel will delete it from that position of the grid order

The table below gives a discretion of each setting

Field

Description

Grid Size

The number of rows and columns that compose the targets game grid

Grid Order

The order in which survivors will be prompted to hit grid targets

Repetitions/Loops

The number of times the grid order pattern will occur in game for each game

Max Extension Threshold

The furthest limit (in mm) of how far away from the survivor the targets will spawn

Min Extension Threshold

The closest distance from the survivor (in mm) in which the targets can spawn

Extension Threshold Increment-Decrement

By how much the maximum extension threshold (in mm) will increase or decrease in response to survivor gameplay

Max games per session

The maximum number of games that can be played for each session (time between login/logout)

Max games per day

The maximum number of games that can be played during an entire day

Interval between session

The minimum number of hours after a session before a survivor can play again

Distance before grid adjusts

The distance (in mm) from any given target the survivor's hand must reach before the countdown begins to bring the target closer

Countdown before grid adjusts

The time (in seconds) before the countdown begins for the target to move closer to the survivor

Make sure Enter a value for each setting. you can use the up and down arrow shown in the below above or you can click on the field you wish to change and type in the desired value.

once you have finished changing the settings of the game click commit changes.
